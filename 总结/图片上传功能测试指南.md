# 图片上传功能测试指南

## 📋 测试前准备

### 1. 确保服务运行
```bash
# 启动后端服务
cd /media/storage/project/aicode/backend
npm start

# 启动前端服务（新终端）
cd /media/storage/project/aicode/frontend/services
node server.js
```

### 2. 访问地址
- 前端: http://localhost:3005
- 后端API: http://localhost:3006

### 3. 测试账号
需要先注册一个测试账号或使用已有账号登录

## 🐛 常见问题排查

### 问题1: 上传按钮不显示
**可能原因**:
- 未登录
- JavaScript加载失败

**解决方法**:
1. 确认已登录
2. 打开浏览器控制台查看错误
3. 刷新页面

---

### 问题2: 上传失败
**可能原因**:
- 后端服务未启动
- 数据库连接失败
- 上传目录权限不足

**解决方法**:
```bash
# 检查后端服务
curl http://localhost:3006/health

# 检查数据库连接
# 查看后端日志

# 检查目录权限
ls -la /media/storage/project/aicode/uploads
chmod 755 /media/storage/project/aicode/uploads -R
```

---

### 问题3: 图片库加载失败
**可能原因**:
- API接口错误
- Token过期

**解决方法**:
1. 打开浏览器开发者工具
2. 查看Network标签的API请求
3. 检查响应状态和错误信息
4. 如果token过期，重新登录

---

### 问题4: 图片不显示
**可能原因**:
- 静态文件服务未配置
- 图片路径错误
- 图片文件被删除

**解决方法**:
1. 检查图片URL是否正确
2. 直接访问图片URL测试
3. 查看服务器uploads目录

---

### 问题5: CORS错误

- CORS配置不正确
