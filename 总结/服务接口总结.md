# Monaco Editor 服务接口总结

## 项目概述

本项目是基于 Microsoft Monaco Editor 的代码编辑器服务，提供完整的代码编辑、语法高亮、自动补全等功能。项目包含多个服务组件，支持二次开发和自定义扩展。

## 服务架构

### 1. 核心服务组件

#### 1.1 基础服务器 (server.js)
- **端口**: 3005
- **协议**: HTTP
- **功能**: 静态文件服务
- **访问地址**: http://localhost:3005
- **默认页面**: demo.html

#### 1.2 网站服务器 (website-server.js)
- **端口**: 3006
- **协议**: HTTP
- **功能**: 生产环境网站服务
- **访问地址**: http://localhost:3006
- **服务目录**: website/dist/

#### 1.3 开发服务器 (webpack-dev-server)
- **端口**: 3007
- **协议**: HTTP
- **功能**: 开发环境热重载
- **访问地址**: http://localhost:3007

### 2. 前端应用结构

#### 2.1 主要页面路由

## API 接口

### 1. 编辑器 API

#### 1.1 基础编辑器操作
```typescript
// 创建编辑器
const editor = monaco.editor.create(container, options);

// 设置值
editor.setValue('new code');

// 获取值
const code = editor.getValue();

// 格式化代码
editor.getAction('editor.action.formatDocument')?.run();

// 设置语言
monaco.editor.setModelLanguage(model, 'javascript');

// 设置主题
monaco.editor.setTheme('vs-dark');
```

#### 1.2 语言服务 API
```typescript
// 注册语言
monaco.languages.register({ id: 'myLanguage' });

// 设置语言配置
monaco.languages.setLanguageConfiguration('myLanguage', config);

// 设置词法分析器
monaco.languages.setMonarchTokensProvider('myLanguage', definition);

// 注册补全提供者
monaco.languages.registerCompletionItemProvider('javascript', {
    provideCompletionItems: (model, position) => {
        return { suggestions: [] };
    }
});
```

### 2. 组件 API

#### 2.1 MonacoEditor 组件
```typescript
interface MonacoEditorProps {
    model: monaco.editor.ITextModel;
    onEditorLoaded?: (editor: monaco.editor.IStandaloneCodeEditor) => void;
    height?: MonacoEditorHeight;
    theme?: string;
    readOnly?: boolean;
    className?: string;
}
```

#### 2.2 ControlledMonacoEditor 组件
```typescript
interface ControlledMonacoEditorProps {
    value: string;
    onDidValueChange?: (newValue: string) => void;
    language?: string;
    theme?: string;
}
```

### 3. 状态管理 API

#### 3.1 MobX 状态管理
```typescript
// 状态类定义
class EditorState {
    @observable code = '';
    @observable language = 'javascript';
    @observable theme = 'vs';

    @action setCode(code: string) {
        this.code = code;
    }

    @action setLanguage(language: string) {
        this.language = language;
    }

    @action setTheme(theme: string) {
        this.theme = theme;
    }
}
```

### 1. 环境准备

### 3. 开发建议

#### 3.1 组件设计
- 使用 TypeScript 进行类型安全
- 遵循 React 最佳实践
- 使用 MobX 进行状态管理
- 保持组件的单一职责

#### 3.2 代码组织
- 按功能模块组织代码
- 使用绝对路径导入
- 保持文件结构清晰
- 添加适当的注释

#### 3.3 性能优化
- 使用 React.memo 优化渲染
- 合理使用 MobX 的 computed 和 action
- 避免不必要的重新渲染
- 使用懒加载减少初始包大小

