# Monaco Editor 项目结构说明

## 📁 项目整体结构

```
aicode/
├── frontend/                    # 前端项目目录
│   ├── demos/                  # 演示页面
│   │   ├── demo.html          # 基础演示页面
│   │   └── ai-demo.html       # AI 代码补全演示
│   ├── static/                 # 静态资源
│   ├── assets/                 # 资源文件
│   ├── config/                 # 配置文件
│   ├── scripts/                # 脚本文件
│   │   └── start.sh           # 启动脚本
│   ├── services/               # 服务文件
│   │   ├── server.js          # 基础服务器
│   │   └── website-server.js  # 网站服务器
│   ├── package.json           # 前端项目配置
│   └── README.md             # 前端说明文档
├── website/                    # 网站项目（Monaco Editor 官方）
│   ├── src/                   # 源码
│   ├── dist/                  # 构建输出
│   ├── package.json           # 网站配置
│   └── webpack.config.ts      # 构建配置
├── src/                        # Monaco Editor 核心源码
├── samples/                    # 官方示例
├── build/                      # 构建工具
├── monaco-lsp-client/          # LSP 客户端
├── webpack-plugin/             # Webpack 插件
├── 总结/                       # 文档总结
│   ├── 二次开发示例/           # 开发示例
│   ├── 服务接口总结.md         # API 文档
│   ├── AI集成总结.md          # AI 功能文档
│   └── 第三方API配置示例.md   # 第三方 API 文档
├── server.js                   # 根目录服务器（重定向）
├── package.json                # 根项目配置
└── README.md                   # 项目说明
```

## 🎯 前端文件组织原则

### 1. 前端文件统一放在 `frontend/` 目录
- **演示页面**: `frontend/demos/`
- **静态资源**: `frontend/static/`
- **服务文件**: `frontend/services/`
- **脚本文件**: `frontend/scripts/`

### 2. 文件分类规则
- **HTML 演示页面**: 放在 `demos/` 子目录
- **静态资源** (CSS, JS, 图片): 放在 `static/` 子目录
- **服务器文件**: 放在 `services/` 子目录
- **启动脚本**: 放在 `scripts/` 子目录

### 3. 访问路径映射
- `/` → `frontend/demos/demo.html`
- `/ai` → `frontend/demos/ai-demo.html`
- `/static/` → `frontend/static/`

## 🚀 启动方式

### 1. 使用根目录服务器（推荐）
```bash
# 在项目根目录
node server.js
# 访问: http://localhost:3005
```

### 2. 使用前端启动脚本
```bash
# 在 frontend 目录
./scripts/start.sh
```

### 3. 使用 npm 脚本
```bash
# 在 frontend 目录
npm start
```

## 📋 服务端口分配

| 服务 | 端口 | 路径 | 说明 |
|------|------|------|------|
| 根目录服务器 | 3005 | `/` | 重定向到前端文件 |
| 网站服务器 | 3006 | `/` | Monaco Editor 官方网站 |
| 开发服务器 | 3007 | `/` | Webpack 开发服务器 |

## 🔧 开发指南

### 添加新的演示页面
1. 在 `frontend/demos/` 创建 HTML 文件
2. 在根目录 `server.js` 中添加路由
3. 更新文档

### 添加静态资源
1. 将文件放入 `frontend/static/`
2. 通过 `/static/filename` 访问

### 修改服务配置
1. 编辑 `frontend/services/` 中的服务文件
2. 更新根目录 `server.js` 中的路由
