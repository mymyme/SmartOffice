## 📝 技术要点

### 1. 正确使用db.query
```javascript
// ✅ 正确：db.query已经解构，直接使用
const rows = await db.query(sql, params);
const firstRow = rows[0];

// ❌ 错误：重复解构
const [rows] = await db.query(sql, params);
```

### 2. MySQL LIMIT语法
```javascript
// ✅ 正确：LIMIT offset, count
LIMIT ?, ?
[offset, limit]

// ❌ 错误：LIMIT ? OFFSET ? 不是标准语法
LIMIT ? OFFSET ?
[limit, offset]
```

### 3. 模块导入解构
```javascript
// auth.js导出
module.exports = {
    authenticate,
    optionalAuthenticate
};

// ✅ 正确：解构导入
const { authenticate } = require('./auth');

// ❌ 错误：直接导入
const authMiddleware = require('./auth');
```

### 4. 保持选中状态
```javascript
// ✅ 支持可选清除
function hideToolbar(clearSelection = true) {
    toolbar.hide();
    if (clearSelection) {
        selectedElement = null;
    }
}

// 打开对话框时不清除
hideToolbar(false);
```
