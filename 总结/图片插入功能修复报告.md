# 图片插入功能修复报告

## 🐛 问题描述

### 问题1: 图片404错误
**错误信息**:
```
GET http://8.152.98.33:8300/uploads/images/user_1/202511/3889a0b06a8f12c4e536f443e71cdfd5.png 404 (Not Found)
```

**原因**: 图片URL使用的是前端地址（8300端口），而实际图片在后端（8301端口）

### 问题2: 选中位置失效
**错误信息**:
```
选中位置已失效，将插入到编辑器末尾
```

**原因**: 打开图片库对话框后，原有的Range选中状态丢失

---

## 🔍 根本原因分析

### 问题1: 图片URL路径错误

#### 图片存储位置
```
后端服务器: http://8.152.98.33:8301
图片路径: /uploads/images/user_1/202511/xxx.png
完整URL: http://8.152.98.33:8301/uploads/images/user_1/202511/xxx.png
```

#### 错误的URL构建
```javascript
// 数据库中存储的是相对路径
selectedImageData.url = "/uploads/images/user_1/202511/xxx.png"

// 直接使用相对路径插入HTML
const imageHtml = `<img src="${selectedImageData.url}" ...>`
// ❌ 结果：浏览器使用当前页面的域名和端口
// → http://8.152.98.33:8300/uploads/images/user_1/202511/xxx.png
// → 404错误（前端服务器上没有这个文件）
```

#### 正确的URL构建
```javascript
// 需要添加后端服务器地址
const backendURL = API_BASE_URL.replace('/api', '');
// backendURL = "http://8.152.98.33:8301"

const fullImageUrl = `${backendURL}${selectedImageData.url}`;
// fullImageUrl = "http://8.152.98.33:8301/uploads/images/user_1/202511/xxx.png"

const imageHtml = `<img src="${fullImageUrl}" ...>`
// ✅ 正确：从后端服务器加载图片
```

---

### 问题2: 选中状态丢失

#### 选中状态的生命周期

**正常流程**:
```
1. 用户在预览区选中文本或元素
   → selectedRange 被保存

2. 点击"插入图片"按钮
   → 打开图片库对话框

3. 选择图片

4. 点击"插入图片"
   → selectedRange 已经失效 ❌
```

#### 为什么会失效？

1. **焦点转移**: 点击对话框时，焦点从iframe转移到主页面
2. **Selection改变**: 浏览器的Selection对象被清除
3. **Range失效**: Range对象失去关联

#### 解决方案

使用`selectedElement`作为备用：
```javascript
// 方案1: 使用savedRange（主要）
if (selectedRange) {
    selectedRange.insertNode(imgElement);
}

// 方案2: 使用selectedElement（备用）
else if (selectedElement && selectedElement.parentNode) {
    // 插入到选中元素的后面
    selectedElement.parentNode.insertBefore(imgElement, selectedElement.nextSibling);
}

// 方案3: 插入到编辑器末尾（兜底）
else {
    insertImageToEditor(imageHtml);
}
```

### 修复2: 优化选中位置插入逻辑

**文件**: `/frontend/demos/visual-editor.html`

**修改位置**: `insertImageAtSelection()` 函数

```javascript
function insertImageAtSelection(imageHtml) {
    // ✅ 检查两种选中方式
    if (!selectedRange && !selectedElement) {
        console.log('选中位置已失效，将插入到编辑器末尾');
        insertImageToEditor(imageHtml);
        updateStatus('⚠️ 选中位置已失效，图片已插入到编辑器末尾');
        return;
    }

    try {
        const previewFrame = document.getElementById('preview-frame');
        const iframeDoc = previewFrame.contentDocument || previewFrame.contentWindow.document;

        // 创建图片元素
        const tempDiv = iframeDoc.createElement('div');
        tempDiv.innerHTML = imageHtml;
        const imgElement = tempDiv.firstElementChild;

        // ✅ 优先使用selectedElement（更稳定）
        if (selectedElement && selectedElement.parentNode) {
            // 插入到选中元素后面
            selectedElement.parentNode.insertBefore(imgElement, selectedElement.nextSibling);
            console.log('图片已插入到选中元素后面');
        }
        // ✅ 其次使用selectedRange
        else if (selectedRange) {
            selectedRange.deleteContents();
            selectedRange.insertNode(imgElement);
            console.log('图片已插入到选中Range位置');
        }

        // 同步到编辑器
        syncPreviewToCode();
        updateStatus('✅ 图片已插入成功');
    } catch (error) {
        console.error('选中位置插入失败:', error);
        insertImageToEditor(imageHtml);
    }
}
```

### 2. 避免选中位置失效

#### 技巧1: 快速操作
```
选中文字 → 立即点击工具栏按钮 → 快速选择图片 → 插入
尽量减少等待时间
```

#### 技巧2: 使用元素选中
```
点击元素（而不是拖选文字）
→ 元素选中更稳定
→ 不容易失效
```

#### 技巧3: 使用右键快捷方式
```
右键点击元素 → 工具栏自动出现
→ 立即点击"插入图片"
→ 选中状态更持久
```
