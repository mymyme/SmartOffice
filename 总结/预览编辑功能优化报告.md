# 预览编辑功能优化报告

## 📋 优化需求

根据用户反馈，需要对预览编辑功能进行以下优化：

1. ✅ 在实时预览中，启用预览编辑后，要可以**选中组件**（不只是文本）
2. ✅ 选中组件后**右键弹出局部修改的浮窗**
3. ✅ 局部"AI编辑"时，输入内容后发送**不要提示"选中内容已失效"**
4. ✅ 不修改其他内容

## 🔧 实现方案

### 1. 增强元素选中功能

#### 新增：点击选中元素
```javascript
// 监听元素点击事件（仅在编辑模式下生效）
iframeDoc.addEventListener('click', handleElementClick);

function handleElementClick(e) {
    // 只在启用预览编辑时才支持元素选中
    const toggleBtn = document.getElementById('toggle-edit-btn');
    if (!toggleBtn || !toggleBtn.textContent.includes('禁用')) {
        return;  // 未启用编辑模式，忽略
    }

    let clickedElement = e.target;

    // 忽略body和html
    if (clickedElement.tagName === 'BODY' || clickedElement.tagName === 'HTML') {
        return;
    }

    // 高亮选中的元素
    highlightElement(clickedElement);

    // 保存选中信息
    selectedElement = clickedElement;
    selectionText = clickedElement.outerHTML;

    // 创建Range对象
    const range = iframeDoc.createRange();
    range.selectNode(clickedElement);
    selectedRange = range;

    // 显示浮动工具栏
    showFloatingToolbar(e);
}
```

#### 新增：元素高亮功能
```javascript
// 高亮选中的元素
function highlightElement(element) {
    clearElementHighlight();  // 清除之前的高亮
    element.classList.add('preview-selection-highlight');
    element.setAttribute('data-selected', 'true');
}

// 清除元素高亮
function clearElementHighlight() {
    const iframeDoc = previewFrame.contentDocument;
    const highlighted = iframeDoc.querySelectorAll('.preview-selection-highlight');
    highlighted.forEach(el => {
        el.classList.remove('preview-selection-highlight');
        el.removeAttribute('data-selected');
    });
}
```

### 2. 右键菜单快捷打开局部编辑

#### 新增：右键菜单监听
```javascript
// 监听右键菜单
iframeDoc.addEventListener('contextmenu', handleContextMenu);

function handleContextMenu(e) {
    // 只在启用预览编辑时才支持
    const toggleBtn = document.getElementById('toggle-edit-btn');
    if (!toggleBtn || !toggleBtn.textContent.includes('禁用')) {
        return;
    }

    e.preventDefault();  // 阻止默认右键菜单

    let clickedElement = e.target;

    // 忽略body和html
    if (clickedElement.tagName === 'BODY' || clickedElement.tagName === 'HTML') {
        return;
    }

    // 高亮选中的元素
    highlightElement(clickedElement);

    // 保存选中信息
    selectedElement = clickedElement;
    selectionText = clickedElement.outerHTML;

    // 创建Range对象
    const range = iframeDoc.createRange();
    range.selectNode(clickedElement);
    selectedRange = range;

    // 直接打开局部编辑对话框
    openLocalEditDialog();
}
```

### 3. 保持选中状态不失效

#### 修改1: 隐藏工具栏时可选择是否清除选中
```javascript
// 修改前
function hideFloatingToolbar() {
    toolbar.classList.remove('show');
    selectedRange = null;        // 总是清除
    selectedElement = null;      // 总是清除
    selectionText = '';          // 总是清除
}

// 修改后
function hideFloatingToolbar(clearSelection = true) {
    toolbar.classList.remove('show');

    // 只有在明确要求清除时才清除选中状态
    if (clearSelection) {
        selectedRange = null;
        selectedElement = null;
        selectionText = '';
        clearElementHighlight();
    }
}
```

#### 修改2: 打开局部编辑对话框时不清除选中
```javascript
function openLocalEditDialog() {
    // ... 其他代码 ...

    // 隐藏工具栏但不清除选中状态
    hideFloatingToolbar(false);  // 传入false保持选中

    // 聚焦输入框
    setTimeout(() => {
        document.getElementById('local-edit-input').focus();
    }, 300);
}
```

#### 修改3: 发送局部编辑请求时不验证选中是否失效
```javascript
async function sendLocalEditRequest() {

## 🎯 功能效果

### 场景1: 点击选中元素

**操作流程**:
```
1. 点击"🖊 启用预览编辑"按钮
2. 在预览区点击任意元素（如按钮、标题、图片等）
3. 元素被蓝色虚线框高亮
4. 浮动工具栏自动出现在元素上方
5. 可以选择：✏️ AI编辑 | 📷 插入图片 | 🎨 样式 | 🗑️ 删除
```

**效果**:
- ✅ 任意元素都可以通过点击选中
- ✅ 选中的元素有明显的视觉反馈（蓝色虚线框）
- ✅ 浮动工具栏智能定位
- ✅ 可以快速进行各种操作

---

### 场景2: 右键快捷编辑

**操作流程**:
```
1. 启用预览编辑
2. 在预览区右键点击任意元素
3. 元素被高亮选中
4. 直接打开局部编辑对话框

**效果**:
- ✅ 右键直接打开编辑对话框，省去点击工具栏步骤
- ✅ 更符合用户习惯（右键=上下文菜单）
- ✅ 提升编辑效率

**操作流程**:
```
1. 选中一个元素（点击或右键）
2. 打开局部编辑对话框
3. 输入多条编辑指令：
   - "改为红色"
   - "加粗文字"
   - "增加内边距"
4. 每次发送后继续输入下一条
5. 无需重新选择元素
```

**效果**:
- ✅ 选中状态一直保持，不会失效
- ✅ 可以连续发送多条编辑指令
- ✅ 不会提示"选中内容已失效"
- ✅ 对话框关闭后选中状态依然保持

## 🐛 已修复的问题

### 问题1: 只能选中文本
**症状**: 用户只能通过鼠标拖选文本来选中，无法选中整个元素

**原因**: 只监听了`mouseup`事件，仅捕获文本选择

**修复**: 新增`click`事件监听，支持点击选中元素

---

### 问题2: 无右键快捷方式
**症状**: 每次编辑都要点击工具栏的按钮，操作繁琐

**原因**: 未实现右键菜单功能

**修复**: 新增`contextmenu`事件监听，右键直接打开编辑对话框

---

### 问题3: 选中内容频繁失效
**症状**: 打开局部编辑对话框后，发送指令提示"选中内容已失效"

**原因**:
1. `hideFloatingToolbar()`总是清除选中状态
2. `sendLocalEditRequest()`总是检查选中是否失效

**修复**:
1. `hideFloatingToolbar(clearSelection=true)` 添加可选参数
2. 打开对话框时传入`false`保持选中
3. 移除发送时的失效检查
