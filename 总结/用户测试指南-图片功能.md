## 🐛 常见问题

### 问题1: 仍然显示500错误
**可能原因**: 浏览器缓存
**解决方案**:
```
1. 按 Ctrl+Shift+Delete 清除缓存
2. 或使用隐私/无痕模式重新访问
3. 或更换浏览器测试
```

### 问题2: 图片库打不开
**可能原因**: Token过期
**解决方案**:
```
1. 退出登录
2. 重新登录
3. 再次测试
```

### 问题3: 上传图片失败
**可能原因**: 文件大小或类型
**解决方案**:
```
1. 确保图片小于10MB
2. 确保是 jpg/png/gif/webp 格式
3. 尝试其他图片
```

## 💡 调试技巧

### 1. 打开浏览器控制台
```
按 F12 打开开发者工具
切换到 Console 标签
查看详细错误信息
```

### 2. 查看网络请求
```
在开发者工具中切换到 Network 标签
点击 XHR 过滤器
执行操作（如点击图片库按钮）
查看请求和响应
```

### 3. 检查请求头
```
在 Network 标签中点击请求
查看 Headers
确认:
- Authorization: Bearer <token> 存在
- Content-Type 正确
```
