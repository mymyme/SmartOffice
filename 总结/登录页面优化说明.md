**修改后：**

```javascript
    return;
}
```

**影响的功能：**

#### 3. 登出功能

**修改前：**

## 可能的缺点

### 1. 用户体验

- ❌ 需要跳转页面（增加一次页面加载）
- ❌ 无法在当前页面快速登录
- ❌ 跳转可能导致编辑内容丢失（需要做好保存提示）

### 2. 解决方案

**自动保存草稿：**
```javascript
// 跳转前保存当前编辑内容
function saveBeforeLogin() {
    const code = editor.getValue();
    localStorage.setItem('draft', code);
    window.location.href = '/demos/login.html';
}

// 登录后恢复
function restoreDraft() {
    const draft = localStorage.getItem('draft');
    if (draft) {
        editor.setValue(draft);
        localStorage.removeItem('draft');
    }
}
```
