## 启动命令

### 基础服务器 (端口3005)
```bash
cd /media/storage/project/aicode
node server.js
# 访问: http://localhost:3005
```

### 网站服务器 (端口3006)
```bash
cd /media/storage/project/aicode
node website-server.js
# 访问: http://localhost:3006
```

### 开发服务器 (端口3007)
```bash
cd /media/storage/project/aicode/website
npm run dev
# 访问: http://localhost:3007
```

## 验证端口可用性

在启动服务前，可以使用以下命令检查端口是否可用：

```bash
# 检查端口3005
netstat -tulpn | grep :3005

# 检查端口3006
netstat -tulpn | grep :3006

# 检查端口3007
netstat -tulpn | grep :3007
```
