# 管理后台重置密码功能使用说明

## 功能说明

管理员可以直接为任何用户重置密码，重置后该用户将被强制退出所有会话，需要使用新密码重新登录。

## 访问地址

- **本地访问：** http://localhost:3005/demos/admin.html
- **远程访问：** http://8.152.98.33:8300/demos/admin.html

## 使用步骤

### 1. 登录管理后台

使用管理员账号登录系统。

### 2. 查找目标用户

在用户列表中找到需要重置密码的用户：

- **浏览列表**：直接在表格中查找
- **使用搜索**：在搜索框中输入用户名或邮箱
- **筛选条件**：可以按角色、状态筛选

### 3. 点击"重置密码"按钮

在用户操作列中，找到**橙色的"重置密码"按钮**（位于"编辑"和"删除"按钮之间）。

### 4. 填写新密码

弹出的重置密码窗口中：

1. **阅读警告提示**
   ```
   ⚠️ 警告
   重置密码后，用户将被强制退出所有会话，需要使用新密码重新登录。
   ```

2. **输入新密码**
   - 密码长度至少8位
   - 建议包含字母、数字和特殊字符

3. **确认密码**
   - 再次输入相同的新密码

### 5. 确认操作

1. 点击**"确认重置"**按钮
2. 在弹出的二次确认对话框中点击**"确定"**
3. 等待操作完成

### 6. 完成

看到**"密码重置成功，用户需要使用新密码重新登录"**提示，表示操作成功。

## 操作效果

### 对用户的影响

重置密码后，目标用户将：

1. ✅ **立即退出登录** - 所有设备上的会话都会被清除
2. ✅ **旧密码失效** - 无法使用原密码登录
3. ✅ **需要新密码** - 必须使用管理员设置的新密码登录

### 日志记录

系统会自动记录两条日志：

1. **管理员操作日志**
   ```
   动作：ADMIN_RESET_PASSWORD
   描述：重置用户密码: testuser (ID: 5)
   操作人：admin
   时间：2025-11-17 17:45:23
   IP地址：192.168.1.100
   ```

2. **用户日志**
   ```
   动作：PASSWORD_RESET
   描述：密码被管理员重置 (管理员ID: 1)
   时间：2025-11-17 17:45:23
   IP地址：192.168.1.100
   ```

## 常见问题

### Q1: 密码有什么要求？

**A:**
- 最少8位字符
- 建议包含字母、数字和特殊字符
- 不超过128位

### Q2: 重置密码后用户能看到新密码吗？

**A:**
- 用户**无法看到**新密码
- 管理员需要**通过其他方式**（如邮件、电话）告知用户新密码
- 建议用户首次登录后立即修改密码

### Q3: 可以重置管理员的密码吗？

**A:**
- 可以，管理员也可以重置其他管理员的密码
- 包括自己的密码（会导致自己被退出登录）

### Q4: 重置后多久生效？

**A:**
- **立即生效**
- 用户的所有现有会话会被立即清除
- 用户需要立即使用新密码重新登录

### Q5: 如果输错密码怎么办？

**A:**
- 可以再次重置
- 没有次数限制
- 每次重置都会记录日志

### Q6: 可以批量重置密码吗？

**A:**
- 目前不支持批量重置
- 需要逐个用户操作
- 未来版本可能会添加此功能

## 安全提醒

### 管理员注意事项

1. **谨慎使用** - 重置密码会影响用户的使用，请确认必要性
2. **告知用户** - 重置后及时通知用户新密码
3. **使用强密码** - 设置的新密码应符合强度要求
4. **及时修改** - 建议让用户首次登录后立即修改密码
5. **注意日志** - 所有操作都会被记录，请勿滥用

### 用户注意事项

1. **首次登录** - 收到新密码后尽快登录
2. **立即修改** - 登录后前往个人中心修改密码
3. **妥善保管** - 不要将密码告诉他人
4. **定期更换** - 建议定期修改密码

## 错误处理

### 常见错误及解决方法

#### 错误1：密码至少需要8个字符

**原因：** 输入的密码太短

**解决：** 输入至少8位字符的密码

#### 错误2：两次输入的密码不一致

**原因：** 新密码和确认密码不匹配

**解决：** 检查两次输入的密码是否完全一致

#### 错误3：用户不存在

**原因：** 目标用户已被删除

**解决：** 刷新用户列表，确认用户是否存在

#### 错误4：需要管理员权限

**原因：** 当前登录账号不是管理员

**解决：** 使用管理员账号登录

#### 错误5：网络错误

**原因：** 网络连接问题或服务器未响应

**解决：**
- 检查网络连接
- 确认后端服务器是否运行
- 稍后重试
