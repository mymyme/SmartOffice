# 图片拖拽调整大小功能说明

## ✨ 功能介绍

在实时预览中，所有插入的图片都可以通过拖拽调整大小，就像在Photoshop或Word中一样！

### 核心特性

1. **8个调整手柄** - 四角+四边中点
2. **保持宽高比** - 按住Shift键拖拽
3. **实时预览** - 拖拽时即时显示大小
4. **同步到代码** - 调整后自动更新编辑器代码
5. **恢复原始大小** - 双击图片恢复
6. **视觉反馈** - 鼠标悬停显示蓝色边框

---

## 🎯 使用方法

### 方法1: 拖拽调整大小

#### 步骤1: 选中图片
```
在预览区点击任意图片
→ 图片周围出现8个蓝色圆点（调整手柄）
→ 图片显示蓝色边框
```

#### 步骤2: 拖拽手柄
```
将鼠标移到任意调整手柄上
→ 光标变为调整大小图标
→ 按住鼠标左键拖拽

8个调整点：
┌───●───┐  ● = 调整手柄
●       ●
├───●───┤  四角：nw, ne, se, sw
●       ●  四边：n, e, s, w
└───●───┘
```

#### 步骤3: 释放鼠标
```
松开鼠标完成调整
→ 自动同步到编辑器
→ 状态栏显示新尺寸
```

---

### 方法2: 保持宽高比

```
按住 Shift 键 + 拖拽任意手柄
→ 图片按比例缩放
→ 不会变形
```

**推荐使用**：
- 照片：始终保持宽高比（按住Shift）
- 图标：可自由调整
- 装饰图：根据需要选择

---

### 方法3: 恢复原始大小

```
双击图片
→ 立即恢复到原始尺寸
→ 移除自定义宽度和高度
```

---

## 📐 调整手柄说明

### 四角手柄（推荐）
```
nw (左上) ┌  ne (右上) ┐
sw (左下) └  se (右下) ┘

功能：同时调整宽度和高度
光标：双向斜箭头 (↖ ↗ ↙ ↘)
用途：最常用，最灵活
```

### 边中点手柄
```
n (上) ↑    s (下) ↓
e (右) →    w (左) ←

功能：只调整单边
光标：单向箭头
用途：精确控制某一维度
```

## 💡 使用技巧

### 技巧1: 等比例缩放
```
按住 Shift 键拖拽
→ 保持图片原始宽高比
→ 适合照片和图标
```

### 技巧2: 精确调整单边
```
使用边中点手柄（n, s, e, w）
→ 只改变一个维度
→ 适合调整横向或纵向尺寸
```

### 技巧3: 快速恢复
```
调整失误？双击图片
→ 立即恢复原始大小
→ 重新开始
```

### 技巧4: 批量调整
```
1. 调整第一张图片到理想大小
2. 记住尺寸（状态栏显示）
3. 依次调整其他图片到相同大小
→ 保持视觉统一
```

### 技巧5: 响应式设计
```
使用百分比宽度：
- 小图：20-30%
- 中图：40-60%
- 大图：80-100%

拖拽到合适大小后，
可在编辑器中手动改为百分比
```

## 🔄 自动同步

### 同步机制

调整大小后，代码会自动更新：

**调整前**:
```html
<img src="image.jpg" alt="图片">
```

**调整后**:
```html
<img src="image.jpg" alt="图片" style="width: 400px; height: 300px;">
```

### 同步时机

1. **拖拽结束时** - 松开鼠标自动同步
2. **双击恢复时** - 立即同步
3. **手动触发** - 点击其他地方

## ⚙️ 技术细节

### 最小尺寸限制
```javascript
最小宽度：50px
最小高度：50px
```

防止图片被缩得太小而看不见。

### 宽高比计算
```javascript
aspectRatio = 原始宽度 / 原始高度

按住Shift时：
- 拖拽左右边 → 高度 = 宽度 / aspectRatio
- 拖拽上下边 → 宽度 = 高度 * aspectRatio
```

### 事件处理
```javascript
mouseenter → 显示蓝色边框
mouseleave → 移除边框（未选中时）
click → 显示调整手柄
dblclick → 恢复原始大小
mousedown → 开始拖拽
mousemove → 实时调整
mouseup → 完成调整，同步代码
```

## 🐛 常见问题

### 问题1: 看不到调整手柄

**原因**: 图片未被选中

**解决**:
```
1. 点击图片
2. 确保图片周围出现蓝色边框
3. 应该看到8个蓝色圆点
```

---

### 问题2: 拖拽时图片变形

**原因**: 没有按住Shift键

**解决**:
```
按住 Shift 键再拖拽
→ 保持原始宽高比
→ 图片不会变形
```

---

### 问题3: 调整后代码没更新

**原因**: 同步延迟或出错

**解决**:
```
1. 点击预览区空白处
2. 或手动点击"应用修改"按钮
3. 或按 Ctrl+S 保存
```

---

### 问题4: 双击没反应

**原因**: 点击间隔太长

**解决**:
```
快速双击图片
→ 两次点击间隔 < 300ms
```

---

### 问题5: 手柄太小难以点击

**解决方案**:
```
手柄尺寸：10px × 10px
边框：2px白色
可点击区域实际更大

技巧：
- 将鼠标移到手柄附近
- 看到光标变化时点击
```
